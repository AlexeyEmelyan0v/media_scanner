## Сборка
```
cmake -S . -B build
cmake --build build
```

## Запуск
Запуск без аргументов:
```
./build/media_scanner
```
В этом случае приложение сканирует домашнюю директорию `$HOME` с интервалом 10 секунд.

Запуск с указанием директории и интервала:
```
./build/media_scanner [директория] [интервал_в_секундах]
```
Результат записывается в файл `.media_files` внутри сканируемой директории.
